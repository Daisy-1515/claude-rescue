## 📊 颜色系统

### 新增颜色
- `brightGreen` (92) - 正常状态、成功消息
- `brightYellow` (93) - 警告、忙碌状态
- `brightRed` (91) - 错误、异常
- `brightCyan` (96) - SessionID、选中箭头
- `brightBlue` (94) - 备用高亮

### 颜色语义
| 元素 | 颜色 | 含义 |
|------|------|------|
| 标题栏 | 反色 + 粗体 | 主区域 |
| 分隔线 | 灰色（dim） | 视觉分组 |
| SessionID | 亮青色 | 唯一标识 |
| 目录路径 | 蓝色 | 位置信息 |
| 正常状态 | 亮绿色 | 健康 |
| 忙碌/重试 | 亮黄色 | 注意 |
| 错误 | 亮红色 | 严重 |
| 成功消息 | 亮绿色 + ✓ | 操作完成 |

---

## 🎯 设计原则

1. **层次清晰** - 使用分隔线和颜色建立清晰的信息层次
2. **对比度** - 关键信息使用明亮色，背景信息使用暗色
3. **一致性** - 交互模式和非交互模式使用相同的配色方案
4. **可访问性** - 保留 `--no-color` 选项，支持无色环境
5. **图标增强** - 适度使用 emoji 图标，增强识别度但不过度
